二、下载整个项目

1、windows
  1. 下载和安装git
  2. 安装完成后，在开始菜单找到 “Git” -> “Git Bash”,会跳出类似命令行窗口的东西，就说明Git安装完成
  3. 复制项目的地址
  4. 通过 git clone 下载仓库
    git clone 项目地址
  5. 或者直接下载zip文件
    1、Mac
  6. 下载和安装homebrew

    通过 homebrew安装git

  7. 下载 git(前提是homebrew安装)
    brew install git
  8. 检查git是否安装成功
    git --version
# 若输出如下内容，说明安装成功
# git version 2.23.0
